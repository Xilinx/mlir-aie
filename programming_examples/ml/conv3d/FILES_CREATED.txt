DESIGN HIGHLIGHTS
================================================================================

Stampable Block Pattern
-----------------------
- Column-based organization (8 columns on NPU2)
- Vertical stacking of cores within columns
- Each column uses independent shim DMA
- Spatial parallelism across height dimension

Scalability
-----------
  Cores | Device    | Layout | Shim DMAs
  ------|-----------|--------|----------
    1   | NPU2Col1  | 1×1    | 1+1
    2   | NPU2Col2  | 2×1    | 2+2
    4   | NPU2Col4  | 4×1    | 4+4
    8   | NPU2      | 8×1    | 8+8
   16   | NPU2      | 8×2    | 8+8
   32   | NPU2      | 8×4    | 8+8

Performance
-----------
  Cores | Expected Speedup | Use Case
  ------|------------------|-------------------
    1   | 1×               | Baseline
    8   | ~7-8×            | Large models
   16   | ~13-15×          | Very large models
   32   | ~24-28×          | Maximum throughput

Key Features
------------
✓ Auto-detects device capabilities
✓ Simple core function (no conditionals)
✓ Parallel shim DMA (up to 8 channels)
✓ TensorAccessPattern for clean slicing
✓ PyTorch validated
✓ Well documented (30+ KB docs)
